Normal weekly use from VS Code
==============================

1. Download weekly-proposal-pack.zip from GitHub Actions.
2. Open this project in VS Code.
3. Open START_HERE/1_import_weekly_proposal_pack.py and click Run Python File.
4. Select the downloaded zip when the file picker opens.
5. Read the scraper status and change summary shown in the terminal.
6. Open START_HERE/2_open_pricing_editor.py and click Run Python File.
7. Review, adjust, save, and export final prices in the editor.

You normally do not need to launch any other files manually.

Double-click option
===================

If you are using Windows Explorer instead of VS Code, you can still double-click:
- 1 Import weekly proposal pack.cmd
- 2 Open pricing editor.cmd
