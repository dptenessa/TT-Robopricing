Normal weekly use
=================

1. Download weekly-proposal-pack.zip from GitHub Actions.
2. Double-click "1 Import weekly proposal pack.cmd".
3. Select the downloaded zip when the file picker opens.
4. Read the scraper status and change summary shown on screen.
5. Double-click "2 Open pricing editor.cmd".
6. Review, adjust, save, and export final prices in the editor.

You normally do not need to launch any other files manually.
