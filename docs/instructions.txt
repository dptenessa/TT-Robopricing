PREP WORK:
I. Define pricing units in inputs/pricing_units.json
II. Define promos in inputs/promos.json
III. Define wholesale countries and costs in inputs/WS_PPG.csv
IV. Optional for GitHub Actions: store WS_PPG.csv as repository secret WS_PPG_CSV_B64 instead of committing it.

NORMAL WORKFLOW:
1. Run `python automation/pricing_pipeline.py --scrape` for the weekly scrape + proposal flow.
2. Run `python "automation/fast pricing editor.py"` to compare proposals against the last export/session and manually correct prices.
3. Save from the editor. The save becomes the new last export and also writes history.

FASTER LOCAL WORKFLOW:
Run `python automation/pricing_pipeline.py` to reuse existing scrapes/*_current.csv files, rebuild combined market data, annotate outliers, and regenerate USD/EUR proposals.

ONE-COMMAND UI WORKFLOW:
Run `python automation/pricing_pipeline.py --scrape --open-editor`.

CANONICAL FILE AREAS:
- scrapes/*_current.csv: raw scraper current files
- outputs/combined_scrapes/: combined competition latest, history, and change reports
- outputs/market_analysis/: market data after outlier annotation
- outputs/model_proposals/USD/ and outputs/model_proposals/EUR/: model proposals by currency
- outputs/manual_prices/: current, autosaved, and historical manual editor prices/promos
- outputs/partner_packs/: default clean partner ZIP location
